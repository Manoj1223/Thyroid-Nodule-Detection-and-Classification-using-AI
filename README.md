# AI-Based Thyroid Nodule Detection and Classification

## 🧠 Project Overview

This project presents an **AI-based system for automated detection and classification of thyroid nodules using ultrasound images**.
The system uses a **Convolutional Neural Network (CNN)** model to analyze thyroid ultrasound images and classify nodules as **benign or malignant**.

A **Flask-based web application** allows users to upload ultrasound images and receive prediction results.

## 🎯 Objectives

* Detect thyroid nodules automatically from ultrasound images
* Classify nodules as **benign or malignant**
* Provide a **web interface** for easy image upload
* Assist doctors with **AI-based diagnosis support**

---

---

## 🔄 System Workflow

1. User uploads a **thyroid ultrasound image**
2. Image is sent to the **Flask backend**
3. Image preprocessing (resize, normalization)
4. Processed image is passed to the **CNN model**
5. Model extracts features from the ultrasound image
6. System predicts **benign or malignant**
7. Result is displayed on the **web application**

---

## 🏗️ System Architecture

User → Upload Image → Flask Backend → Image Preprocessing → CNN Model → Classification → Result Display

---

## 📊 Model Information

* Model Type: **Convolutional Neural Network (CNN)**
* Input: **Ultrasound Image**
* Output: **Benign or Malignant Classification**
